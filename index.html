C# OOP Basics
The course (OOP) examines how to work with classes and objects, how to use object-oriented modeling, and build class hierarchies. The basic principles of OOP that are studied are abstraction (interfaces, abstract classes), encapsulation, inheritance and polymorphism.
C# OOP Advanced
The course (OOP) examines the principles and best practices for building quality software. Concepts for writing High Quality Code in C #. Includes the use of abstractions, processing of bad code (refactoring), and writing of unit tests and reflections. The fundamentals of SOLID Principles.

C# Databases

MS SQL
The course introduces us to one of the most widely used database management systems (DBMS) used in the development of modern information systems - Microsoft SQL Server. The course focuses on relational modeling, data modeling with ER diagrams (tables and relational links) and SQL language (data retrieval, selection, projection, joining, aggregation, grouping, modification, deletion and insertion).
Entity Framework Core
The course tracks in detail the ORM technology: Entity Framework Core (EF core), which is an ORM standard in C # and .NET Core applications. EF Core allows mapping between a relational base and an object-oriented model using both the "database first" and "code first" approaches and provides a powerful object-oriented API for querying the database and performing CRUD operations.

C# Web

Web Basics

ASP.NET Core
The ASP.NET Core course introduce how to build modern web applications with the Model-View-Controller architecture using HTML5 , databases , Entity Framework Core and other technologies. The ASP.NET Core Technology Platform , its components and architecture , the creation of MVC web applications , the definition of models , views and partial views with Razor view engine , the definition of controllers , the operation of databases and integration with Entity Framework Core , LINQ and S QL Server . The course also includes more sophisticated themes, such as working with users , roles and sessions , using AJAX , caching , web application security , web sockets, and working with libraries from MVC controls . Includes several practical lab exercises (lab)and workshops to build comprehensive, full-featured ASP.NET Core web applications.
